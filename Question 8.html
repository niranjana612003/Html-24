ARRAY
An array is a collection of homogenous data items stored in continuous memory location.
SEARCHING ARRAY
searching is the process of looking for a specific element in an array.
LINEAR SEARCH
the linear search approach compares the key element,key with each element in the array.
BINARY SEARCH
For binary search to work, the elements in the array must already be ordered.
STACK
A pile of objects, typically one that is neatly arranged
